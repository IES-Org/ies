Overview

Welcome to the IES Ontologies documentation. This site provides comprehensive documentation for our collection of RDF ontologies designed to standardize information exchange.

Components

  • IES Common: Core concepts and relations
  • IES Build: The built environment
  • IES Infrastructure: Infrastructure in the built environment
  • IES NSD:Security concepts
  • IES People: People, organizations, and roles
  • IES Systems: Systems, integrity, and resilience
