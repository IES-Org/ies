Overview

Welcome to the IES Ontologies documentation. This collection of RDF ontologies provides standardized vocabularies and semantics for information exchange across various domains and systems.

Available Ontologies

IES Common

Core concepts and shared vocabulary that forms the foundation of the Information Exchange Standard. Available in versions 4.0 and 5.0.

IES Build

Standardized vocabulary for describing build processes and deployment configurations. Currently at version 1.0.

IES Infrastructure

Concepts and relationships for describing infrastructure components and resource allocations. Currently at version 1.0.

IES Network Service Discovery

Vocabulary for describing service discovery mechanisms and network service capabilities. Currently at version 1.0.

IES People

Definitions for describing people, organizations, roles, and responsibilities. Currently at version 1.0.

IES Systems

Vocabulary for describing system components, their relationships, and interactions. Currently at version 1.0.
